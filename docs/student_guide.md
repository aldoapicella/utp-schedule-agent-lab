# Student Guide

1. Create and activate your virtual environment for your OS.
2. Run `python -m scripts.tasks init-env`
3. Run `python -m scripts.tasks setup`
4. Run `python -m scripts.tasks seed`
5. Start the API with `python -m scripts.tasks run-api`
6. Start the web UI with `python -m scripts.tasks run-web`
7. Explore tests and complete stage TODOs on `student-start`
8. Use `python -m scripts.tasks list-stages` to navigate the class material by stage.
