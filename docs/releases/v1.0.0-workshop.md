# v1.0.0-workshop

Release inicial del laboratorio docente **UTP Schedule Agent Lab**.

## Incluye

- Core determinista de cálculo de horarios UTP reutilizado en `src/schedule_calculator`.
- Capa agentic en `src/schedule_agent` con tools, orquestación, memoria, validación, monitoreo, seguridad y colaboración humana.
- API en FastAPI y dashboard web en Next.js.
- Flujo multiplataforma para Windows, macOS y Linux mediante `python -m scripts.tasks`.
- Material didáctico por stages con documentación, diagramas Mermaid y comandos de ejecución.
- Ramas docentes:
  `student-start`
  `instructor-solution`
- Tags pedagógicos:
  `stage-00-core`
  `stage-01-design`
  `stage-02-tools`
  `stage-03-orchestration`
  `stage-04-memory`
  `stage-05-validation`
  `stage-06-monitoring`
  `stage-07-security`
  `stage-08-human-collaboration`
  `stage-09-web`
  `stage-09-capstone`
